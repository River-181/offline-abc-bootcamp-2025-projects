# Vibe Coding 신규 애플리케이션 10선 - One-Shot 프롬프트 가이드

## 1. 🎲 운세 룰렛 & 타로 카드
**난이도**: ⭐⭐ (2시간)  
**핵심 기능**: 애니메이션 룰렛, 카드 뒤집기, 결과 해석

### 예시 프롬프트

```
단일 HTML 파일로 운세 룰렛과 타로 카드 웹앱을 만들어주세요.

메인 기능:
1. 화면 중앙에 화려한 룰렛 (12개 섹션)
2. 각 섹션: 오늘의 운세, 행운의 숫자, 컬러, 조언 등
3. "돌리기" 버튼으로 룰렛 회전 (3-5초)
4. 회전 중 클릭음과 속도 감소 효과
5. 결과 모달창에 상세 운세 표시

타로 카드 모드:
1. 22장의 메이저 아르카나 카드
2. 카드 뒷면 표시 후 3장 선택
3. 선택한 카드 3D 플립 애니메이션
4. 과거-현재-미래 해석 제공
5. 각 카드의 의미와 조합 해석

특수 효과:
1. 룰렛 회전 시 배경 블러와 스포트라이트
2. 당첨 섹션 깜빡임과 파티클 효과  
3. 타로 카드 선택 시 신비로운 광채
4. 배경에 은은한 별 애니메이션
5. 결과 공유 버튼 (이미지로 저장)

데이터:
1. 오늘 날짜별로 다른 운세 생성
2. 방문 횟수 카운트 및 표시
3. 지난 운세 히스토리 (최근 7개)
4. 즐겨찾기 운세 저장 기능

디자인:
- 신비로운 보라색/금색 테마
- 그라데이션과 네온 효과
- 점성술 기호와 별자리 장식
- 모바일 최적화 (세로 모드)
```

## 2. 📊 실시간 투표 시스템
**난이도**: ⭐⭐⭐ (2.5시간)  
**핵심 기능**: 투표 생성, 실시간 결과, 공유 기능

```
단일 HTML 파일로 실시간 투표 시스템을 만들어주세요.

투표 생성:
1. 질문과 2-6개 선택지 입력
2. 투표 유형: 단일선택/복수선택
3. 마감 시간 설정 (선택사항)
4. 익명/기명 투표 옵션
5. 투표 생성 시 고유 ID 부여

투표 참여:
1. 선택지 클릭으로 간단 투표
2. 투표 변경 가능 옵션
3. 투표 전 결과 숨김 설정
4. 참여자 수 실시간 표시
5. 남은 시간 카운트다운

결과 시각화:
1. 막대 그래프로 득표율 표시
2. 실시간 애니메이션 업데이트
3. 파이 차트 토글 옵션
4. 각 옵션별 투표자 목록 (기명 시)
5. 1위 옵션 왕관 아이콘과 효과

공유 기능:
1. 투표 링크 복사 (URL에 ID 포함)
2. QR 코드 생성 및 표시
3. 결과 이미지로 다운로드
4. SNS 공유 버튼
5. 임베드 코드 제공

관리 기능:
1. 진행 중인 투표 목록
2. 투표 수정/삭제 (생성자만)
3. 실시간 참여 알림
4. 투표 복제 기능
5. localStorage 활용 데이터 저장

디자인:
- 모던한 카드 UI
- 선택 시 바운스 애니메이션
- 그래프 증가 시 부드러운 전환
- 다크/라이트 테마
- 반응형 레이아웃
```

## 3. 🎨 픽셀 아트 에디터
**난이도**: ⭐⭐⭐⭐ (3.5시간)  
**핵심 기능**: 그리기 도구, 레이어, 애니메이션

### 예시 프롬프트

```
단일 HTML 파일로 픽셀 아트 에디터를 만들어주세요.

캔버스 설정:
1. 16x16, 32x32, 64x64 그리드 선택
2. 격자선 표시/숨김 토글
3. 줌 인/아웃 (마우스 휠)
4. 캔버스 이동 (스페이스+드래그)
5. 픽셀 미리보기 창 (실제 크기)

그리기 도구:
1. 펜: 클릭/드래그로 픽셀 그리기
2. 지우개: 픽셀 제거
3. 스포이드: 색상 추출
4. 페인트통: 영역 채우기
5. 선/사각형/원 도구
6. 대칭 모드 (가로/세로/대각선)

색상 관리:
1. 기본 색상 팔레트 (16색)
2. 커스텀 색상 추가/삭제
3. 그라데이션 생성기
4. 최근 사용 색상 히스토리
5. 색상 팔레트 저장/불러오기

레이어 시스템:
1. 최대 5개 레이어 지원
2. 레이어 추가/삭제/복제
3. 불투명도 조절
4. 레이어 순서 변경
5. 레이어 병합

애니메이션:
1. 프레임 추가 (최대 10프레임)
2. 프레임 복사/삭제
3. 재생 속도 조절 (FPS)
4. 미리보기 재생/정지
5. GIF 내보내기

추가 기능:
1. 실행 취소/다시 실행 (Ctrl+Z/Y)
2. PNG 스프라이트 시트 내보내기
3. 프로젝트 저장/불러오기 (JSON)
4. 단축키 안내 모달
5. 참조 이미지 반투명 오버레이

디자인:
- 다크 테마 기본
- 플로팅 툴 패널
- 드래그 가능한 창
- 픽셀 호버 시 좌표 표시
```

## 4. 🧮 암산 스피드 게임
**난이도**: ⭐⭐ (2시간)  
**핵심 기능**: 난이도 조절, 시간 제한, 랭킹

### 예시 프롬프트

```
단일 HTML 파일로 암산 스피드 게임을 만들어주세요.

게임 모드:
1. 시간 공격: 60초 동안 최대한 많이
2. 정확도 모드: 실수 3번이면 종료
3. 레벨 모드: 난이도 점진적 상승
4. 연습 모드: 시간 제한 없음
5. 일일 챌린지: 매일 같은 문제 세트

문제 유형:
1. 기본 사칙연산 (+, -, ×, ÷)
2. 두 자리 수 ~ 네 자리 수
3. 분수 계산 (레벨 5 이상)
4. 제곱근 (레벨 8 이상)
5. 연속 계산 (예: 5+3×2)

게임 플레이:
1. 문제가 화면 상단에 크게 표시
2. 숫자 패드로 답 입력
3. 엔터키로 제출
4. 정답/오답 즉시 피드백
5. 연속 정답 시 콤보 점수

효과 및 피드백:
1. 정답: 초록색 체크와 +점수 애니메이션
2. 오답: 빨간색 X와 화면 흔들림
3. 콤보: 불꽃 효과와 배수 표시
4. 시간 부족: 경고음과 빨간 테두리
5. 신기록: 축하 애니메이션

통계 및 진행도:
1. 현재 점수와 최고 점수
2. 정확도 백분율 표시
3. 평균 응답 시간
4. 레벨별 진행도 바
5. 일일/주간/월간 통계 그래프

부가 기능:
1. 사운드 on/off
2. 진동 피드백 (모바일)
3. 랭킹 시스템 (로컬)
4. 성취 뱃지 시스템
5. 친구와 점수 공유

디자인:
- 깔끔한 미니멀 디자인
- 큰 폰트로 가독성 확보
- 파스텔 색상 사용
- 숫자 입력 시 버튼 애니메이션
```

## 5. 🌸 가상 정원 가꾸기
**난이도**: ⭐⭐⭐ (3시간)  
**핵심 기능**: 식물 성장, 상호작용, 계절 변화

### 예시 프롬프트

```
단일 HTML 파일로 가상 정원 가꾸기 앱을 만들어주세요.

정원 시스템:
1. 5×5 격자의 화분 배치
2. 빈 화분 클릭으로 씨앗 심기
3. 6종류 꽃 선택 가능
4. 각 꽃마다 다른 성장 속도
5. 최대 4단계 성장 (씨앗→새싹→꽃봉오리→만개)

관리 활동:
1. 물주기: 화분 클릭 시 물뿌리개 애니메이션
2. 햇빛: 구름 드래그로 햇빛 조절
3. 비료: 성장 속도 2배 부스트
4. 잡초 제거: 랜덤 등장하는 잡초 클릭
5. 해충 퇴치: 나타나는 벌레 클릭

성장 메커니즘:
1. 실시간 성장 (1분 = 1일)
2. 물 부족 시 시들기 시작
3. 과도한 물은 뿌리 썩음
4. 계절별 성장 속도 차이
5. 인접 꽃들의 상호 영향

계절 시스템:
1. 봄/여름/가을/겨울 자동 순환
2. 계절별 배경과 날씨 변화
3. 특정 계절에만 피는 꽃
4. 겨울엔 온실 모드 필요
5. 계절 이벤트 (벚꽃, 단풍 등)

수집 요소:
1. 꽃 도감 (발견한 꽃 기록)
2. 희귀 색상 변이 (1% 확률)
3. 완벽한 꽃 사진 저장
4. 나비/벌 방문 기록
5. 월간 정원 사진 앨범

보상 시스템:
1. 꽃 판매로 코인 획득
2. 코인으로 새 씨앗/도구 구매
3. 레벨업과 정원 확장
4. 업적 달성 시 특별 아이템
5. 일일 보너스 (연속 접속)

디자인:
- 귀여운 픽셀 아트 스타일
- 부드러운 성장 애니메이션
- 계절별 색감 변화
- 앰비언트 자연 소리
- 줌 인/아웃 기능
```

## 6. 💬 AI 캐릭터 채팅
**난이도**: ⭐⭐⭐ (2.5시간)  
**핵심 기능**: 캐릭터 생성, 대화 시뮬레이션, 감정 표현

### 예시 프롬프트

```
단일 HTML 파일로 AI 캐릭터와 대화하는 채팅 앱을 만들어주세요.

캐릭터 설정:
1. 5가지 사전 정의 캐릭터 (로봇, 마법사, 해적, 외계인, 고양이)
2. 각 캐릭터별 고유 말투와 성격
3. 커스텀 캐릭터 생성 옵션
4. 캐릭터 아바타와 이름 설정
5. 성격 특성 슬라이더 (친절함, 유머, 진지함)

대화 시스템:
1. 사용자 입력에 자연스러운 응답
2. 이전 대화 문맥 기억 (5턴)
3. 캐릭터별 특수 반응과 캐치프레이즈
4. 감정 키워드 인식 및 반응
5. 타이핑 애니메이션 효과

감정 표현:
1. 6가지 기본 감정 (기쁨, 슬픔, 화남, 놀람, 사랑, 피곤)
2. 대화 내용에 따른 표정 변화
3. 이모티콘과 액션 텍스트 (*웃음*, *윙크*)
4. 감정 게이지 시각화
5. 특별한 상황 반응 (생일, 날씨 등)

상호작용 기능:
1. 가위바위보 미니게임
2. 간단한 수수께끼 내기
3. 오늘의 운세 말해주기
4. 노래 가사 따라 부르기
5. 그림 그리기 요청 (ASCII 아트)

대화 관리:
1. 대화 내역 저장/불러오기
2. 좋아하는 대화 북마크
3. 대화 내보내기 (텍스트 파일)
4. 대화 초기화 옵션
5. 여러 캐릭터와 그룹 채팅

UI/UX:
1. 메신저 스타일 인터페이스
2. 캐릭터 프로필 사이드바
3. 다크/라이트 테마
4. 메시지 도착 알림음
5. 스티커/이모지 전송

디자인:
- 부드러운 말풍선 애니메이션
- 캐릭터별 테마 색상
- 타이핑 인디케이터
- 모바일 최적화 레이아웃
```

## 7. 🎪 리액션 타임 챌린지
**난이도**: ⭐⭐ (2시간)  
**핵심 기능**: 반응속도 측정, 다양한 미니게임, 기록 관리

### 예시 프롬프트

```
단일 HTML 파일로 반응속도를 테스트하는 미니게임 모음을 만들어주세요.

게임 모드:
1. 색상 변화: 화면이 초록색 되면 클릭
2. 타겟 히트: 랜덤 위치 타겟 빠르게 클릭
3. 패턴 기억: 순서대로 패턴 따라하기
4. 숫자 찾기: 1부터 순서대로 클릭
5. 반대로 하기: 지시와 반대 행동

색상 변화 게임:
1. 빨간 화면에서 대기
2. 랜덤 시간 후 초록색 변환
3. 반응 시간 밀리초 단위 측정
4. 너무 빠르면 "부정 출발!" 경고
5. 5회 평균 기록 계산

타겟 히트:
1. 화면에 원형 타겟 등장
2. 크기와 위치 랜덤
3. 타겟 클릭 시 다음 즉시 등장
4. 30초간 최대한 많이 클릭
5. 정확도와 속도 점수 계산

기록 시스템:
1. 각 게임별 최고 기록
2. 일일/주간/전체 랭킹
3. 개인 기록 그래프
4. 평균 반응속도 통계
5. 향상도 추적

난이도 설정:
1. 초급/중급/고급 모드
2. 타겟 크기 조절
3. 시간 제한 변경
4. 방해 요소 on/off
5. 연습 모드

특수 효과:
1. 완벽한 타이밍 시 황금 이펙트
2. 연속 성공 시 콤보 표시
3. 신기록 시 폭죽 애니메이션
4. 실패 시 화면 흔들림
5. 카운트다운 사운드

부가 기능:
1. 친구와 점수 비교
2. 일일 챌린지 미션
3. 반응속도 향상 팁
4. 컬러 테마 변경
5. 집중력 향상 모드
```
